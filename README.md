# Simple Calculator (Streamlit)

A calculator web app built with Streamlit.

## Features

- Basic arithmetic operations:
  - Addition (`+`)
  - Subtraction (`-`)
  - Multiplication (`*`)
  - Division (`/`)
  - Floor Division (`//`)
  - Modulo (`%`)
- Input fields in side-by-side columns
- Division/modulo by zero validation
- Rounded result for normal division (`/`) to 2 decimals
- Calculation history (newest first)
- Clear history button
- Reuse last result as next first input

## Tech Stack

- Python
- Streamlit
